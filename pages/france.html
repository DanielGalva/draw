If you want to know how to draw a flag using HTML and CSS, take a look at these tutorials

France Flag

This is what goes on HTML:

    <div>
        <div class="blue vertical"></div>
        <div class="white vertical"></div>
        <div class="red vertical"></div>
    </div>

This is what goes on CSS:

    .vertical {
        width: 75px;
        height: 150px;
        float: left;
    }
    
    .blue {
        background-color: blue;
    }
    .red {
        background-color: red;
    }
    .white {
        background-color: white;
    }
